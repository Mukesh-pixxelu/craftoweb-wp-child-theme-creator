# CraftoWeb WP Child Theme Creator

A lightweight WordPress plugin to create child themes with one click.

## Features

- Create child themes instantly
- Auto-generate style.css
- Auto-generate functions.php
- Copy parent theme screenshot
- Auto-fill child theme details
- One-click activation
- Secure with WordPress nonce verification

## Installation

1. Download the plugin.
2. Upload it to `/wp-content/plugins/`.
3. Activate the plugin.
4. Go to **Child Theme Creator** in the WordPress admin menu.

## Requirements

- WordPress 6.0+
- PHP 7.4+
